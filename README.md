# Small Improvements (small-improvements)

Small Improvements is a lightweight performance management platform designed for teams of 50 to 750 employees, providing structured tools for continuous feedback, performance reviews, OKRs, 1:1 meetings, pulse surveys, praise, and 360-degree feedback. The platform offers a REST API (v2) with over 150 resource endpoints covering users, feedback cycles, review questionnaires, objectives, meetings, surveys, and administrative configuration, authenticated via personal Bearer access tokens.
